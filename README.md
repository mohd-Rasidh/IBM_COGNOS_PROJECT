# 🚚 SwiftRoute Fleet Operations Dashboard

## 📊 Project Overview

This project analyzes logistics and delivery performance using IBM Cognos Analytics.

## 🎯 Objectives

* Monitor shipment performance
* Analyze SLA compliance
* Identify breakdown patterns
* Improve fuel efficiency insights

## 🛠 Tools Used

* IBM Cognos Analytics
* CSV Dataset

## 📈 Features

* Interactive Dashboard
* KPI Metrics
* Driver Performance Analysis
* AI-driven Exploration Insights

## 📂 Files Included

* Dashboard Screenshots
* Reports (PDF)
* Dataset (CSV)

## 📌 Conclusion

This dashboard helps logistics managers make data-driven decisions to improve delivery efficiency.
